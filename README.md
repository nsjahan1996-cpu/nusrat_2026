# Modern Slider / Carousel

A modern, responsive **image/content slider (carousel)** built using **HTML, CSS, and JavaScript**.  
This slider features smooth animations, auto-slide support, and is easy to customize for any website or landing page.

## Features

- Modern & clean UI design
- Smooth slide transition animations
- Auto-slide with configurable timing
- Next / Previous navigation buttons
- Dot / pagination indicators
- Fully responsive (mobile, tablet & desktop)
- Lightweight – no heavy libraries required

## How to Use

1. Download or clone the repository
2. Open the project folder in **VS Code** or any code editor
3. Open `index.html` in your browser  
   - Recommended: use **Live Server** extension in VS Code
4. The slider will start automatically
5. Use navigation arrows or dots to change slides manually
